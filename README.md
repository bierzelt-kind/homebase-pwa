# 🏠 HomeBase

Ein schnelles, hübsches Dashboard für deinen Home Server — als PWA installierbar!

## ✨ Features

- 📱 **PWA** — Installierbar wie eine native App (iOS & Android)
- 🎨 **Schönes Design** — Animated Gradient, Glassmorphism
- 🔄 **Lokal/VPN Toggle** — Schnell zwischen IPs wechseln
- ⚡ **Schnell** — Keine Dependencies, reines HTML/CSS/JS
- 📴 **Offline** — Service Worker für Offline-Nutzung

## 📲 Installation

### iPhone
1. Safari öffnen → Dashboard laden
2. Teilen-Button → "Zum Home-Bildschirm"

### Android
1. Chrome öffnen → Dashboard laden
2. "Zum Startbildschirm hinzufügen"

## ⚙️ Konfiguration

Klicke auf **⚙️ Einstellungen** und trage deine IPs ein:
- **Lokale IP** (z.B. 192.168.0.76)
- **Tailscale/VPN IP** (z.B. 100.86.200.106)

Wird automatisch im Browser gespeichert (localStorage).
