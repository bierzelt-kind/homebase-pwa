# 🏠 HomeBase v2

Dashboard für deinen Home Server — als PWA installierbar!

## ✨ v2 Features
- 🔄 **Live-Status** — Prüft automatisch ob Dienste laufen
- 🌓 **Dark/Light Mode** — Toggle in der Ecke
- 📊 **Status-Bar** — Zeigt wie viele Services online sind
- 🔔 **Auto-Refresh** — Prüft jede Minute
- ➕ **8 Services** — Jellyseerr, Jellyfin, SABnzbd, Sonarr, Radarr, Glances, Portainer, Traefik

## 🚀 Services
| Service | Port | Beschreibung |
|---------|------|-------------|
| 🔍 Jellyseerr | 5055 | Media Requests |
| ▶️ Jellyfin | 8096 | Media Player |
| ⬇️ SABnzbd | 8080 | Downloader |
| 📺 Sonarr | 8989 | TV Series |
| 🎬 Radarr | 7878 | Movies |
| 📊 Glances | 61208 | System Info |
| 🐳 Portainer | 9000 | Docker Management |
| 🌐 Traefik | 8081 | Reverse Proxy |

## 📲 Installation
1. Browser öffnen → URL laden
2. Teilen-Button → "Zum Home-Bildschirm"
3. Fertig! 🎉

## ⚙️ Konfiguration
Einstellungen → IPs eintragen → Speichern
